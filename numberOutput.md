# Swift Number Output

## SwiftNumberOutput

Print numeric values with `print()` and format using interpolation or
format styles.

## Printing Numbers
Print values directly or embed them using string interpolation with `\(value)`

```
let a =7, b=3
print(a,b)
separated by default
print("a=\(a), b=\(b)")
interpolation
print("sum = \(a+b)")
```
